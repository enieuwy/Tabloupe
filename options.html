Performance

Frees memory. Tabs reload when you return to them. Pinned, active, and audio-playing tabs are never discarded.

Tab search shortcut

Opens the tab search overlay on web pages. Click the box and press the keys you want (include at least one of Ctrl, Alt, or Cmd).

The overlay can't run on privileged pages (about:, add-ons, PDF viewer). For a shortcut that works there too, assign one under about:addons → Manage Extension Shortcuts.

Keyboard shortcuts

Lens slots 1–9, cycle next/previous, and show-all commands can be bound in about:addons → ⚙️ → Manage Extension Shortcuts. They ship unbound to avoid conflicts.
